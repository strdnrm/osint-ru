О проекте

Правовая база и принципы работы

Правовая основа

  • ФЗ №229-ФЗ — ФССП: публичный банк данных исполнительных производств
  • ФЗ №129-ФЗ — ФНС: открытые сведения ЕГРЮЛ/ЕГРИП
  • ФЗ №262-ФЗ — ГАС Правосудие: тексты судебных решений
  • ФЗ №127-ФЗ — Федресурс: открытые сообщения о банкротстве
  • НК РФ, ст. 84 — сведения об ИНН (общедоступная часть)

Что не делает

  • Не обращается к закрытым базам данных
  • Не использует утечки и нелегальные базы
  • Не нарушает ФЗ №152 «О персональных данных»
  • Не хранит введённые данные и загруженные фото

Стек

  • Go Бэкенд, параллельные запросы через sync.WaitGroup
  • embed Статика встроена в бинарник — 1 файл
  • multipart Приём фото через форму, base64 в памяти
  • VK API Публичный поиск людей без токена
