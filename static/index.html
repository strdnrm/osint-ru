Правовая основа

Все источники данных — исключительно публичные государственные реестры, доступ к которым обеспечивается федеральными законами РФ:

  • ФЗ №229-ФЗ — «Об исполнительном производстве» (ФССП, банк данных ИП)
  • ФЗ №129-ФЗ — «О государственной регистрации юридических лиц и ИП» (ЕГРЮЛ/ЕГРИП)
  • ФЗ №262-ФЗ — «Об обеспечении доступа к информации о деятельности судов» (ГАС Правосудие)
  • ФЗ №127-ФЗ — «О несостоятельности (банкротстве)» (Федресурс)
  • НК РФ, ст. 84 — Сведения об ИНН (общедоступная часть)

Что не делает этот инструмент

  • Не обращается к закрытым базам данных
  • Не использует утечки и нелегальные базы
  • Не нарушает ФЗ №152 «О персональных данных»
  • Не хранит введённые пользователем данные
  • Не выполняет запросы без явного действия пользователя

Технический стек

  • Go Бэкенд, параллельные запросы к API
  • HTML/CSS/JS Фронтенд, без фреймворков
  • embed Статика встроена в бинарник
  • sync.WaitGroup Конкурентный поиск по всем источникам
